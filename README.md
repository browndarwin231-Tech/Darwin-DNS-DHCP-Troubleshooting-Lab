# Darwin DNS & DHCP Troubleshooting Lab

## Overview

This project demonstrates a common Windows 11 Help Desk troubleshooting scenario involving DNS resolution and DHCP lease issues.

The lab documents the troubleshooting workflow using built-in Windows networking tools to verify connectivity, renew IP configuration, inspect DNS cache, and validate network functionality.

---

## Skills Demonstrated

- Windows 11 Administration
- Help Desk Troubleshooting
- DNS Troubleshooting
- DHCP Troubleshooting
- Command Prompt
- Network Diagnostics
- TCP/IP Configuration
- Documentation

---

## Tools Used

- Windows 11
- Command Prompt
- PowerShell
- ipconfig
- nslookup
- ping
- tracert

---

## Lab Steps

1. Review Network Configuration
2. Verify IP Configuration
3. Test DNS Resolution
4. Test Internet Connectivity
5. Trace Network Route
6. Display DNS Cache
7. Flush DNS Cache
8. Release and Renew DHCP Lease
9. Resolution Summary
